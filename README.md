# MyMusic--Swift
一个学习swift写的简单的音乐项目

功能：

    1.在线

    2.下载、本地

    3.分享、第三方登陆


技术要点：

    1.利用MPMoviePlayerController封装一套音乐播放器（Song.Swift、SongLrc.Swift、SongPlay.Swift）

    2.解析Lrc文件实现歌词同步

    3.动画的交互（pop,UIViewAnimation,LiquidFloatingActionButton）

    4.SnapKit、Alamofire、FMDB、shareSDK等第三方的使用
